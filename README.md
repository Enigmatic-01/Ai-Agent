# AI Air Quality & Weather Agent

This project is an AI-powered assistant that fetches **real-time air pollution data** for Indian cities and provides **current weather information**. It combines **LangChain**, **OpenAI GPT**, **DuckDuckGo search**, and the **OpenWeather API** to answer complex queries about pollution and weather.

---

## Features

- Identify the **most polluted city in India** in real-time (based on PM2.5 or AQI).  
- Fetch **current weather conditions** for any city: temperature, humidity, and general description.  
- Uses a **ReAct agent** framework for intelligent reasoning and tool selection.  
- Supports **DuckDuckGo search** and **API-based weather retrieval**.  

---

## Technologies Used

- **Python 3.x**  
- **LangChain** – For AI agent orchestration and tool integration  
- **OpenAI GPT (ChatOpenAI)** – Natural language reasoning  
- **DuckDuckGoSearchRun** – Web search tool  
- **OpenWeather API** – Real-time weather data  
- **ReAct Agent** – Agent framework combining reasoning and action
